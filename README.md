# Bluelief's dotfiles


## How to setup

```shell
$ sudo apt update && sudo apt -y upgrade
$ sudo apt install git curl
$ mkdir $HOME/package
$ cd $HOME/package
$ git clone https://github.com/bluelief/dotfiles.git
$ cd dotfiles
$ ./unlink && ./install.sh && setup.sh
```
